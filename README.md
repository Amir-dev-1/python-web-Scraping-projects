# Python Web Scraping Projects

This repository contains practical web scraping projects written in Python.
The goal is to collect, process, and store data from websites efficiently.

## Features
- Scraping with requests and BeautifulSoup
- Scraping dynamic websites using Selenium
- Export data to CSV
- Clean and reusable code structure

## Technologies
- Python 3
- Requests
- BeautifulSoup
- Selenium

## How to Run
`bash
pip install -r requirements.txt
python scraper_basic.py
